Leaflet

A Modern, Lightweight Open-Source JavaScript Library for Interactive Maps by CloudMade

Home | Features | Documentation | Examples | Download (zip) | Source on GitHub | Follow on Twitter

Leaflet doesn't try to do everything for everyone. Instead it focuses on making the basic things work perfectly. It still satisfies the needs of the vast majority of map apps developers while being flexible enough to be easily extended by writing third-party plugins.

Available map layers

  • Tile layers
  • Polylines, polygons, circles
  • Image overlays
  • Markers
  • Popups

Interaction features

On desktop browsers

  • Dragging
  • Scroll wheel zoom
  • Double click zoom
  • Shift-drag zoom to bounding box

On mobile browsers (iOS, Android)

  • Touch dragging
  • Multi-touch zoom (using two fingers)

Visual features

  • Panning animation
  • Zooming animation
  • Smooth zoom on mobile browsers
  • Very nice default design for markers and popups

Customization features

  • Customizable marker icons
  • Pure CSS3 popups for easy restyling
  • A simple interface for implementing custom map layers
  • Ability to implement custom map projections (other than the usual Spherical Mercator)

Performance features

  • Hardware acceleration on iOS to make it feel as smooth as native apps
  • Smart polyline/polygon rendering to make it responsive even when displaying objects with many thousands of points
  • Modular build system to reduce the size of the library by leaving out the code you don't need

Browser support

On desktop

  • Firefox 3.6+
  • Chrome
  • Safari 5+
  • IE 7-9
  • IE 6 (not perfect but accessible)

On mobile

  • Safari for iOS 3/4+
  • WebKit for Android 2.2+
